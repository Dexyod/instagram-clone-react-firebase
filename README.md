# Instagram Clone

Built with React and Firebase

## Deployed Build

Build deployed @

(https://instagram-clone-6ebf0.web.app/)

## Features

This Instagram-clone features:

- Firebase Authentication

- Sign Up, Sign In and Logout

- Instagram Embedding

- Upload Images with Captions

- Comment on Posts
